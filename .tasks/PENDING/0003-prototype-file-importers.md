## Acceptance Criteria

- `.eml` import works.
- `mbox` import works.
- `maildir` import works.
- Thunderbird, Evolution, Apple Mail, PST, OLM, and OST support paths are documented.
- Unsupported or partially supported formats produce clear import errors.
