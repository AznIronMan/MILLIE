# Confirm Foundation Decisions

Status: ACTIVE

## Goal

Confirm the core project decisions before implementation begins.

## Context

The project is in planning mode. Current assumptions are SQLite first, HTTP in dev, easy HTTPS/TLS enablement, web/API listeners on `0.0.0.0`, and git disabled for now.

## Acceptance Criteria

- MILLIE acronym is accepted or replaced.
- First implementation stack is chosen.
- Initial schema direction is accepted.
- Attachment storage strategy is chosen.
- Export fidelity goals and first export formats are confirmed.
- Dev/prod auth and network expectations are confirmed.
- Git policy is confirmed before any commit or push workflow starts.
